- **Impact/Why**: Saves small business owners (barbers, dentists, salons) hours spent on phone calls so they can focus on their craft instead of administrative work‚Äîany non-technical owner can deploy a full receptionist in under 2 minutes
- **Languages, Technologies**: Next.js 14, Supabase (Auth + Database), Teli AI (Voice Agent + Phone Provisioning), TypeScript, React

### RentAI (MHacks 2025, https://devpost.com/software/rent-ai)
- **Brief**: AI-powered rental search platform using Fetch.ai agents and Gemini to match users with properties and automate landlord communications via AgentMail
- **Impact/Why**: Streamlines the overwhelming rental search process by eliminating repetitive tasks like browsing endless listings and sending follow-up emails‚Äîbuilt to solve our own off-campus housing search
- **Languages, Technologies**: React, Next.js, TypeScript, Python FastAPI, Fetch.ai, Gemini API, AgentMail API
